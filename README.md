# Industrial Anomaly Detection

This project focuses on detecting anomalies in industrial sensor and production datasets.

In manufacturing environments, early detection of abnormal behavior is critical to prevent equipment failures and production disruptions.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Statistical analysis

## Project Objectives

- Analyze industrial sensor data
- Identify abnormal patterns
- Detect potential equipment failures
- Support predictive maintenance strategies

## Example Techniques

- Statistical threshold detection
- Z-score anomaly detection
- Rolling mean monitoring
- Time-series analysis

## Use Case

Industrial data streams can reveal early signals of process deviations.  
This project demonstrates simple techniques to monitor production data and detect abnormal conditions.
